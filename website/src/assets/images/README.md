## Image Specifications
- **Format**: JPG or PNG
- **Quality**: High resolution for web (1200px width minimum)
- **Optimization**: Compress for web delivery
- **Aspect Ratio**: 16:9 for hero images, 4:3 for apartment images
- **File Size**: Maximum 500KB per image

## Usage
Images are referenced in the Angular application using relative paths from the assets folder. The website includes automatic fallback to placeholder images if specific images are not available.
